# 🏋️ GymMetric

O **GymMetric** é uma aplicação web voltada para criação e gerenciamento de treinos personalizados. O projeto tem como foco ajudar usuários a organizarem suas rotinas de treino de forma prática, com uma interface intuitiva e funcional.

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando:

- **React** – Interface moderna e dinâmica  
- **TypeScript** – Tipagem estática para maior segurança no código  
- **Node.js** – Backend da aplicação  
- **PostgreSQL** – Banco de dados relacional para armazenamento das informações  

## 📌 Funcionalidades

- Criação de listas de treino personalizadas  
- Organização de exercícios por categorias  
- Interface amigável para montagem de treinos  
- Estrutura preparada para futuras funcionalidades (como recomendações automáticas e treinos pré-montados)  

## 🎯 Objetivo

O GymMetric foi desenvolvido com o objetivo de auxiliar usuários na criação e organização de seus treinos, proporcionando mais controle e eficiência durante a prática de atividades físicas.

## 🔧 Estrutura do Projeto

O projeto segue uma arquitetura separada entre frontend e backend:

- **Frontend:** React + TypeScript  
- **Backend:** Node.js  
- **Banco de Dados:** PostgreSQL  

## 📈 Possíveis Melhorias Futuras

- Sistema de recomendações de treino  
- Treinos pré-definidos (hipertrofia, força, emagrecimento)  
- Autenticação de usuários  
- Dashboard com métricas de desempenho  

## 🛠️ Como Executar o Projeto

# Clone o repositório
```bash
git clone https://github.com/Cjr-pjs/GymMetric.git
```

# Acesse a pasta
```bash
cd gymmetric
```

# Instale as dependências
```bash
npm install
```
# Configure as variáveis de ambiente (.env)
```bash
DATABASE_URL=your_database_url

```

# Rode o backend
```bash
npm run dev
```

# Rode o frontend (caso separado)
```bash
npm start
```
